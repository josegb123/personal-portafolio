Sistema Unificado de Gestión M&K

Punto de Venta (POS), Inventario y Cartera Integrados.

El Problema

Celulvariedades M&K manejaba su inventario, ventas y cuentas por cobrar de forma manual o con herramientas separadas (Excel), lo que generaba inconsistencias en el stock, pérdida de tiempo en la conciliación de ventas a crédito, y demoras en el servicio al cliente.

La Solución

Desarrollé un sistema web modular y unificado. El sistema permite registrar ventas rápidamente, actualizar el inventario automáticamente y gestionar la cartera de clientes (créditos) en tiempo real desde una sola plataforma. Esto garantizó datos fiables y operaciones más rápidas.

Tecnologías Utilizadas

PHP (API/Backend) MariaDB HTML5/CSS3 Bootstrap 5 JavaScript
Ver Código en GitHub

Módulos Principales y Beneficios

Punto de Venta (POS)

Facturación rápida con lector de código de barras, cálculo automático de impuestos y descuentos, y emisión de recibos.

Gestión de Inventario

Control de stock en tiempo real, alertas de bajo inventario y seguimiento de entradas/salidas para prevenir pérdidas.

Control de Cartera

Registro de ventas a crédito, seguimiento de abonos, cálculo de saldos pendientes y reportes de morosidad por cliente.

Reportes y Análisis

Generación de reportes de ventas diarias, semanales y mensuales, análisis de productos más vendidos y rendición de cuentas simplificada.
